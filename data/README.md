# data/

Local staging area for datasets. Not checked into git (see `.gitignore`) —
too large and not ours to redistribute.

Download instructions per dataset are in [`../docs/DATASETS.md`](../docs/DATASETS.md).

Suggested layout once you pull something down:

```
data/
├── citrus_leaves/       # Rauf et al. 2019 dataset
├── sweet_orange/        # Bangladesh sweet orange dataset
└── raw_web_images/      # ad-hoc sweet lime images pulled from the web
```
